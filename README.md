# AI Resume Analyzer 

## Features
- Resume parsing (PDF/DOCX)
- AI-based similarity scoring
- Skill extraction
- Missing skill detection
- Suggestions

## Tech Stack
- FastAPI
- Python
- Sentence Transformers
- HTML + Tailwind CSS

## How to Run
1. Start backend:
   uvicorn main:app --reload

2. Start frontend:
   python -m http.server 5500
